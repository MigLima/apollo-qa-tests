# Apollo Project - QA Documentation

Este repositório contém artefatos de testes realizados no projeto Apollo, uma aplicação mobile.

## 🧪 Minha atuação

Atuei na validação de funcionalidades e fluxos da aplicação, com foco em qualidade e usabilidade.

Principais atividades:
- Execução de testes manuais
- Identificação e reporte de bugs
- Validação de fluxos de usuário
- Testes exploratórios e funcionais

## 📌 Funcionalidades testadas

- Login
- Recuperação de senha
- Navegação entre telas
- Validação de formulários

- BUG_REPORTS.md → Bugs identificados
